# Laravel8
Laravel8版の画像掲示板です
laravelで作った掲示板はネットの記事に良くありますが、
laravel8の記述でプログラミングしているサイトは少ない印象があり、今回git hub で更新しました。
# コマンド
```sh
"npm run development",
"development": "mix",
"watch": "mix watch",
"watch-poll": "mix watch -- --watch-options-poll=1000",
"hot": "mix watch --hot",
"prod": "npm run production",
"production": "mix --production"
```
